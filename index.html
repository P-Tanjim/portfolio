I loves turning complex ideas into simple, beautiful, and functional interfaces. I work with HTML, CSS, JavaScript, Node.js, and MongoDB, and I’m currently learning React and MySQL to grow as a full-stack developer. I’m passionate about performance, modern UI design, and creating interactive web experiences that feel smooth and alive.
